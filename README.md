# GTS.Conect - Ecossistema de Gestão de Alta Performance

O **GTS.Conect** é uma plataforma de gestão empresarial integrada, projetada para oferecer visibilidade total sobre operações, clientes, leads e saúde financeira. Com uma interface sofisticada e técnica, o sistema foca em clareza de dados e eficiência operacional.

## 🚀 Módulos do Sistema

### 📊 Dashboard Estratégico
Visão geral e em tempo real dos principais indicadores de desempenho (KPIs), permitindo uma tomada de decisão rápida e baseada em dados reais.

### 💼 Gestão de Projetos
Controle completo de entregas, prazos e equipes.
- Acompanhamento de progresso percentual.
- Gestão de estágios (briefing, design, desenvolvimento, etc.).
- Vinculação de funcionários e clientes.

### 🎯 CRM & Pipeline de Leads
Gestão do funil de vendas desde o primeiro contato até o fechamento.
- Acompanhamento de status de prospecção.
- Histórico de leads e conversão.

### 💰 Fluxo de Ativos (Financeiro)
Interface analítica premium para controle de liquidez e recebíveis.
- **Gráficos de Performance:** Comparativo entre projeção e faturamento liquidado.
- **Conciliação Analítica:** Gestão de parcelas e status de inadimplência.
- **KPIs Financeiros:** Total recebido, valores em aberto e médias mensais.

### 👥 Gestão de Clientes e Equipe
Base de dados centralizada para clientes e colaboradores, facilitando a comunicação e atribuição de responsabilidades.

## 🛠️ Tecnologias Utilizadas

- **Framework:** Next.js 15+ (App Router)
- **Estilização:** Tailwind CSS v4 (Midnight Technical Theme)
- **Banco de Dados & Autenticação:** Firebase (Firestore & Auth)
- **Visualização de Dados:** Recharts (Composed Charts, Area Charts, Pie Charts)
- **Animações:** Framer Motion
- **Ícones:** Lucide React

## 🎨 Filosofia de Design
O sistema utiliza uma abordagem **"Técnica e Profissional"**, priorizando fontes serifadas para legibilidade editorial, fontes mono para dados precisos e transições suaves para uma experiência de usuário de alto nível.
